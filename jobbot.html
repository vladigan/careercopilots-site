Three tiers. (1) Memory — JobBot remembers how you answered the same question last time. (2) Overrides — you can set fixed answers for any question pattern (e.g. "How many years of Python?" → 7). The fallback isn't a black box: every LLM answer is logged so you can override it for next time.
